# Criando Ambiente Virtual
```shell
python3 -m venv .venv 
```
# Ativando Ambiente Virtual
```shell
source .venv/bin/activate
```
# Cirando migrações
```shell
python3 manage.py makemigrations 
```
# Criando Super Usuario
```shell
python3 manage.py createsuperuser
```
# Subindo o Servidor
```shell
python3 manage.py runserver
```
